OBS动态模糊设置教程 | 让画面变自然的小技巧(๑•̀ㅂ•́)و✧

Hello～想让直播/录屏里的游戏跑动、画面切换更丝滑？(｡•̀ᴗ-)✧ 动态模糊就能搞定！这篇教程从版本要求到参数调试，还加了Minecraft的效果对比，手把手教你给OBS加“温柔滤镜”，新手也能轻松上手~

适用场景：游戏直播、摄像头录制、教程录屏；
难度等级：入门级，核心参数直接抄作业就行~

一、版本&插件准备 (｡•̀ᴗ-)✧

1. OBS版本要达标哦

  • 最低门槛：OBS Studio 27.0（低于这个版本装不了插件啦(´･ω･`)）；
  • 推荐版本：OBS Studio 30.0+（插件更乖，不容易卡~）；
  • 特殊提示：用NVIDIA模糊的话，得更到OBS 31.0+才行！

2. 插件二选一（都是免费的！）

两款插件各有优点，按自己需求挑就好~

插件名称 适合谁用 可爱优势 下载链接
StreamFX 新手/直播党 界面简单，不卡屏~ 点我下载StreamFX
Composite-blur 录屏/细节控 模糊效果更自然，能精准控范围~ 点我下载Composite-blur

二、插件安装三步走 (≧▽≦)

1. 先关OBS哦！

安装前把所有OBS窗口关掉，不然插件会加载不上~

2. Composite Blur安装（全系统通用）

  1. 下对应系统的Composite-blur压缩包；
    下载对应系统版本：

    • Windows：obs-composite-blur-1.5.2-windows-x64.zip
    • macOS：`obs-composite-blur-1.5.2-macos-universal.zip一定要看好macOS版本再下载!
    • Linux：obs-composite-blur-1.5.2-linux-x86_64.tar.gz

  2. . 安装步骤（Windows）

    1. 关闭 OBS Studio必须完全退出

    2. 解压下载的 ZIP 文件，得到data和obs-plugins两个文件夹

    3. 复制到 OBS 安装目录：

      • 默认路径：C:\Program Files\obs-studio\
      • 覆盖同名文件夹确认权限，选择 “合并” 而非 “替换”

    4. 验证安装：

      • 启动 OBS → 点击帮助 → 日志文件 → 查看日志

      • 在文件夹里找到最近的日志文件使用自带的记事本
        ‘ctrl+f搜索composite-blur，可查询并看到obs-composite-blur.dll即成功

    3. 安装步骤（macOS）

    1. 关闭 OBS，解压 ZIP 文件必须完全退出
    2. 打开 Finder → 按住Option键点击前往 → 资源库
    3. 进入Application Support/obs-studio/
    4. 创建plugins和data文件夹（如不存在）
    5. 复制obs-composite-blur.plugin到plugins文件夹
    6. 复制data/composite-blur到data文件夹
    7. （macOS 14+）如遇安全提示，需在系统设置→隐私与安全性中允许插件运行

    4. 安装步骤（Linux）

    1. 关闭 OBS，解压 tar.gz 文件必须完全退出

    2. 终端执行：
      bash
      运行
      mkdir -p ~/.config/obs-studio/plugins mkdir -p ~/.config/obs-studio/data cp -r obs-composite-blur/bin/64bit/libobs-composite-blur.so ~/.config/obs-studio/plugins/ cp -r data/composite-blur ~/.config/obs-studio/data/

    3. 启动 OBS 验证

看到「Composite-blur」就成功啦(๑•̀ㅂ•́)و✧！

3. StreamFX安装

步骤和上面差不多 解压到OBS插件文件夹，重启后找「StreamFX」就行~

三、动态模糊怎么调？(๑•́₃•̀๑)

1. 给来源加滤镜

  1. 选中OBS里要模糊的内容（比如游戏、摄像头）；
  2. 右键→「滤镜」，点「+」加对应滤镜：
    • Composite-blur：选「Composite Blur」，再把类型改成“运动模糊”~
    • StreamFX：选「StreamFX > 模糊 > 动态模糊」；

2. 核心参数抄作业

Composite-blur

以Composite-blur动态模糊插件为例，这些是实测适配度最高的参数~(๑•̀ㅂ•́)و✧

参数 推荐值（游戏） 推荐值（屏幕录制） 小提示
模糊算法 时间模糊 时间模糊 时间模糊也就是动态模糊啦 录屏选时间模糊更丝滑~
模糊强度 0.6-0.7 0.5-0.6 游戏别超0.7哦，不然战斗画面会看不清(´･ω･`)
清理阈值 0.2-0.3 0.1-0.2 数值越低，模糊边缘越干净，录屏文字多的话选0.1更清晰~
合成的背景源 无（全屏时） 场景背景源 录屏时选“场景背景源”，能避免模糊覆盖到前景内容~
效果蒙版 无（全屏） 裁剪（局部） 游戏全屏模糊就选”无”，录屏只模糊背景的话选”裁剪”框出区域

StreamFX

以StreamFX动态模糊插件为例

参数 功能说明 推荐值（游戏） 推荐值（屏幕录制） 小提示
Blur Type 模糊类型 emporal Blur Temporal Blur 选这个类型的模糊最自然，不会有“糊成一片”的问题~
Blur Amount 模糊强度 0.6-0.7 0.5-0.6 游戏别超0.7哦，不然战斗画面会看不清(´･ω･`)
Cleanup Threshold 清理阈值 0.2-0.3 0.1-0.2 数值越低，模糊边缘越干净，录屏文字多的话选0.1更清晰~
Algorithm 模糊算法 Multi-Pass Box Gaussian 游戏选Multi-Pass Box更省性能，录屏选Gaussian更丝滑~
Background Source 背景源 无（全屏时） 场景背景源 录屏时选“场景背景源”，能避免模糊覆盖到前景内容~
Mask Type 蒙版类型 None（全屏） Rectangle（局部） 游戏全屏模糊就选None，录屏只模糊背景的话选Rectangle框出区域~

四、不同场景的小优化 (｡•̀ᴗ-)✧

1. 游戏直播（以Minecraft为例）

  • 插件：Composite-blur；
  • 对应参数：Blur Type选Temporal Blur + Blur Amount设0.65 + Algorithm选Multi-Pass Box；
  • 小技巧：配合“盒式模糊”算法，跑图时方块拖影柔和又不卡~

Minecraft有无动态模糊的效果对比

在Minecraft这类方块格游戏里，动态模糊的差异会特别直观~
❌ 无动态模糊的情况：

  • 跑图/飞鞘翅时，方块场景“跳帧”明显，快速转视角会显得生硬卡顿；
  • 挖方块、挥剑战斗的动作轨迹断开，像“瞬间切换”，视觉连贯性差；
  • 多人联机时，队友快速跑跳的轨迹很“碎”，容易跟丢目标。

✅ 开启动态模糊的情况：

  • 跑图/飞鞘翅时，方块场景带轻微拖影，视角转动过渡像“顺滑动画”；
  • 挖方块、挥剑的动作有柔和轨迹拖影，战斗动作更连贯自然；
  • 队友移动轨迹更清晰，即使快速跑跳也不会显得“跳来跳去”。

2. 摄像头录制

  • 插件：StreamFX；
  • 对应参数：Blur Amount设0.5 + Mask Type选Rectangle框出背景区域；
  • 小技巧：加“场景背景源”当背景，只模糊身后的画面，脸蛋全程清清楚楚~

3. 教程录屏

  • 插件：随便选；
  • 对应参数：Blur Amount设0.55 + Cleanup Threshold设0.1 + Algorithm选Gaussian；
  • 小技巧：用Rectangle蒙版框出除了文字/操作区之外的区域，既丝滑又不影响看教程~

五、踩坑&优化小Tips

1. 卡屏怎么办？

  • 把Algorithm换成Multi-Pass Box，同时降低Blur Amount到0.6；
  • 开“GPU加速”（OBS设置→输出→选自己显卡的编码器）；
  • 把预览窗口缩成1/2，会流畅很多~

2. 常见问题速解

问题 解决办法
模糊覆盖到文字/人脸 调整Mask Type为Rectangle，框出只模糊的区域~
模糊边缘很“脏” 降低Cleanup Threshold数值，比如调到`0.1~
插件找不到 升级OBS到30.0+，重新解压插件到正确文件夹~

总结

调OBS动态模糊其实超简单 选对版本和插件，抄好上面的参数就能让画面变自然啦(๑•̀ㅂ•́)و✧！尤其是Minecraft这类游戏，开了动态模糊后跑图、战斗的视觉体验会提升不少 有问题欢迎在评论区交流，插件都是免费的，记得从官网下载哦~
